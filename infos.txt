----------- INIT et DEPOT INITIAL ---------
git init 
voir la config : git config -l

pour ajouter une entrée de config
git config --global user.name 'Antoine Boury'

pour remplacer une entrée existante 
git config --global --replace-all user.name 'Antoine Boury'

pour retier une entrée 
git config --global --unset user.name

-- relier le depot local au depot distant 
git remote add origin https://github.com/antoineby/projet.git

-- ajouter les fichiers au projet local
git add --all 

-- commiter les fichiers du projet local 
git commit -m "Depot initial"

-- Envoi sur le depot distant 
git push -u origin master 

------ GIT CREATION DE BRANCHE ------
git branch recette 
-- on copie le projet sur la branche locale recette 
git checkout recette 

git branch -- pour voir les branches et l'étoile * indique la branche courante 

-- on fait nos modifs 
git add --all (tout rajouter)
git add page.php (ajouter un fichier précis)

git commit -a -m "Premiere mise à jour"
git push -u origin recette 

COMMIT
-a all prise en compte des fichiers modifiés et supprimés 
-m "message" libéllé du commit 

PUSH
-u (set upstream) alimenter le suivi en cas de succès du push
